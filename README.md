OPCToModbus(OPC2Modbus)

An OPC-DA to Modbus TCP Gateway

OPCToModbus is a lightweight, portable gateway that converts data from traditional OPC DA Servers into a Modbus TCP Server.
The software is fully portable (no installation required) and supports Windows XP, Windows 7, Windows 10, both 32-bit and 64-bit.

The free version supports up to 100 Modbus register mappings.
For extended limits, please contact us to obtain a professional license.

Features:
1. Supported OPC DA Data Types
OPCToModbus can read and convert the following OPC DA data types:
    Boolean
    Char / Byte
    Short / Word
    Long / DWord
    Float
    Double (automatically converted to Float)

2. Supported Modbus Function Codes
    Code	Description
    01	Read Coils
    02	Read Discrete Inputs
    03	Read Holding Registers
    04	Read Input Registers
    05	Write Single Coil
    06	Write Single Register
    15	Write Multiple Coils
    16	Write Multiple Registers
    22	Mask Write Register
    23	Read/Write Registers
3. Multi-OPC DA Server Support

Connect to multiple OPC DA Servers simultaneously

Automatic reconnection on connection loss

4. Modbus TCP Server

Supports up to 16 simultaneous TCP client connections

Displays all active client sessions

Automatically disconnects idle clients after 60 seconds

5. System Tray Mode

The gateway can run minimized in the Windows system tray.

6. Event Window & Logging

All runtime information and events appear in the event window

Optional log output to a file

7. CSV Import

Supports importing tag configuration from CSV files.

8. Project File Support

Save the entire configuration as a project file

Automatically load the most recent project at startup
